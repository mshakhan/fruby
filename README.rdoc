== Description

Some functional programming features for ruby.

== Usage

Now implemented only *defun* method. You can use it like this:

  module Math
    class << self
      include FRuby::Definer
      defun(:fact, Fixnum) { |n| n * fact(n - 1) }
      defun(:fact, 0) { 1 }
      defun(:fact, 1) { 1 }
    end
  end

  Math::fact(5) #=> 120
